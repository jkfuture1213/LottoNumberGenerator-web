사용 방법

  1. 'Generate' 버튼을 누르면 세 줄로 로또 번호가 자동으로 생성됩니다.
  2. 첫 번째 줄은 45개의 숫자를 숫자 5개 단위로 끊어 생성합니다.
  3. 두 번째 줄은 3개 단위, 세 번째 줄은 1개 단위로 끊어 생성합니다.
  4. 3개의 줄 중 하나를 선택해 로또를 하시면 됩니다.
  5. 'Generate' 버튼을 다시 누르면 재생성됩니다.
  6. 'Reset' 버튼을 누르면 모든 번호가 초기화됩니다.
